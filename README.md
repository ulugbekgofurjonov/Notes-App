## ✨ Features

- 📱 **Fully Responsive** - Works on all devices (mobile, tablet, desktop)
- 🌓 **Dark Mode** - Beautiful dark theme with auto-save
- 💾 **Auto Save** - Notes saved automatically in browser
- ✏️ **Edit & Delete** - Full CRUD functionality
- ⚡ **Lightning Fast** - Pure vanilla JS, zero dependencies
- 🔒 **Privacy First** - All data stays in your browser

## 💻 Usage

1. **Create Note**: Click "+ Add Note" button
2. **Edit Note**: Click "Edit" button on any note
3. **Delete Note**: Click "Delete" button and confirm
4. **Toggle Theme**: Click 🌙/☀️ icon in header
